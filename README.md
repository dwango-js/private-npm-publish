# private-npm-publish

a command line tool to prevent accidental npm publish.

This command is wrapper of `npm publish`.

- Check valid private module?
- Run `npm publish`

## Installation

    npm install -g private-npm-publish

## Usage

    $ private-npm-publish
    # is the same `npm publish`
